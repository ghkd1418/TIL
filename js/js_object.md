
- 설명이 필요하지 않은 데이터 리스트들은 <b>array</b>
- 설명이 필요한 정보가 담긴 데이터 리스트들은 <b>object</b>

object.function 형식 만드는법

```js
const player = {
	name: "nico",
	point: 10,
	fat: true,
};

console.log(player);
player.point = false;
console.log(player);
```
